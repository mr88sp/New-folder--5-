# Soheili Wood - فروش تخصصی MDF و ورق کابینت 🪵

<div dir="rtl" align="right">

## 🌟 معرفی پروژه
پروژه **Soheili Wood** یک وب‌سایت شرکتی فروشگاهی برای فروش انواع MDF، ورق کابینت و محصولات مرتبط با صنعت چوب می‌باشد. این پروژه با استفاده از **Astro** و **TypeScript** توسعه داده شده است و از کامپوننت‌های **React** برای بخش‌های تعاملی استفاده می‌کند.

### ✨ ویژگی‌های اصلی
- ✅ طراحی کاملاً ریسپانسیو و واکنش‌گرا
- ✅ انیمیشن‌های روان و زیبا با Framer Motion
- ✅ سئو شده و بهینه برای موتورهای جستجو
- ✅ پشتیبانی از تصاویر WebP و AVIF
- ✅ استایل‌دهی با Tailwind CSS
- ✅ تایپ‌سازی کامل با TypeScript
- ✅ آماده اتصال به پنل مدیریت
- ✅ فونت فارسی وزیرمتن

### 📱 صفحات پروژه
1. **صفحه اصلی** - معرفی برند و محصولات ویژه
2. **محصولات** - لیست کامل محصولات با فیلتر
3. **جزئیات محصول** - اطلاعات کامل هر محصول
4. **استعلام قیمت** - جداول قیمت محصولات
5. **رنگ‌بندی MDF** - کاتالوگ رنگ‌ها و پالت‌ها
6. **پروژه‌ها** - گالری پروژه‌های اجرا شده
7. **درباره ما** - تاریخچه، تیم، افتخارات
8. **تماس** - اطلاعات تماس و فرم ارتباطی

## 🚀 شروع سریع

### پیش‌نیازها
- Node.js نسخه 18 یا بالاتر
- npm نسخه 9 یا بالاتر

### نصب و اجرا
```bash
# 1. کلون کردن پروژه
git clone https://github.com/soheiliwood/soheili-wood.git
cd soheili-wood

# 2. نصب وابستگی‌ها
npm install

# 3. اجرا در محیط توسعه
npm run dev:astro

# 4. ساخت نسخه نهایی
npm run build:astro
npm run start:astro
